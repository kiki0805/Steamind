About

Steamind is a tool for visualizing games from your own Steam library as well as other games from Steam that might suit your interests. It is a final project for the course DH2321 Information Visualization given at KTH Royal Institute of Technology. By displaying the games that you own, Steamind aims to provide a way for you to gain insight of your own gaming preferences. Furthermore, a handful of other games from Steam are shown based on your top recommended tags.

The visualization consists of a force-directed graph organized in a hierarchical structure with two levels. Starting at the root node which is the user, four branches representing game categories emerges from it. At the very top, we have the leaves of the tree which are the games.

The Data

The data was retrieved through Steam's API and through web-scraping. The nature of data is nominal, numerical, and categorical. The data is stored in a database through SQLite.

Learning Objectives

Our main learning objectives for this project was to learn to work with Javascript and Javascript libraries. We used d3.js and jQuery. We also created our own API which we used to gather, and filter, the data dynamically. This was done with Python, Scrapy, Flask, and Peewee.
